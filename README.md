# DevHub Homebrew Tap

Official Homebrew tap for [DevHub](https://hubdev.io) — Hybrid Development Environment Manager.

## Installation

```bash
brew tap hubdev-io/tap
brew install --cask devhub
```

## Update

```bash
brew upgrade --cask devhub
```

## Uninstall

```bash
brew uninstall --cask devhub
```
